Во время работы в университете активно использовался язык программирования С для моделирования физических экспериментов. В частности, основной темой работы было построение компьютерной модели сенсора Шека-Хартмана, применяющегося для анализа формы поверхности волнового фронта.

Компьютерная модель сенсора была нужна для моделирования гартманограмм стандартных оптических аберраций.

Форма волнового фронта задавалась функцией z(x,y) в виде произвольной строки (например, z=cos(x)+x*y). Потом делался разбор выражения и быстрое преобразование Фурье (FFT). Программа разрабатывалась на С в популярном в то время IDE - Borland C++ Builder 5. Исходный код проекта и скриншот можно посмотреть на GitHub.
